How to deploy on GitHub Pages

1. Create a new GitHub repo (e.g. outreach-tracker)
2. Upload this HTML file as index.html
3. Go to Settings → Pages, set source to main branch
4. Visit yourusername.github.io/outreach-tracker
5. Paste your GitHub token in settings — data syncs via Gist automatically

Data

All contacts are stored in a private GitHub Gist (when token is set) and also cached in localStorage as a fallback. Clearing your token does not delete your Gist data.
